Conditional Aggregation

SUMIF / AVERAGEIF / MAXIFS / MINIFS
=SUMIF(B2:B11,"MUMBAI",C2:C11)
=SUMIF(B2:B11,">=10000")
=SUMIFS(B2:B15,C2:C15,"PUR",D2:D15,">=15")

Logic

IF / IFS
=IF(A2>=40,"PASS","FAIL")
=IF(AND(A2>=40,B2>=40),"PASS","FAIL")
=IF(OR(A2>=40,B2>=40),"PASS","FAIL")
=IF(F2>=60,"FIRST",IF(F2>=45,"SECOND",IF(F2>=40,"PASS","FAIL")))
=IFS(F2>=60,"FIRST",F2>=45,"SECOND",F2>=40,"PASS",F2<40,"FAIL")
IFS() replaces nested IF — no need to repeat IF keyword; still write every condition including the last

Lookup & Reference

VLOOKUP / HLOOKUP / LOOKUP
VLOOKUP — searches first column; table must be ascending
=VLOOKUP(E2,A2:B9,2,0)
HLOOKUP — searches top row; same logic
=HLOOKUP(B2,$B$12:$D$13,2)*C2
LOOKUP — single row/column vector
=LOOKUP(D2,$A$2:$A$4,$B$2:$B$4)*E2
Range lookup FALSE = exact match only
MATCH / INDEX
MATCH(val, array, 0) → position of exact match
MATCH(val, array, 1) → largest ≤ val (ascending data)
MATCH(val, array, -1) → smallest ≥ val (descending data)
=MATCH("PUNE",A1:A15,0)
INDEX(array, row, col) → value at intersection
=INDEX(A1:F15,7,5)
Combine MATCH+INDEX for flexible lookups on large data

String Functions

Extract & Transform
LEFT("COMPUTER",3) → COM
RIGHT("COMPUTER",3) → TER
MID("COMPUTER",4,3) → PUT
LEN("COMPUTER") → 8
UPPER("computer") → COMPUTER

SUBSTITUTE — by text match

SUBSTITUTE with instance_num replaces only that occurrence

Financial Functions

FV / PV / PMT
FV(rate,nper,pmt,[pv]) — future value
PV(rate,nper,pmt,[fv]) — present value
PMT(rate,nper,pv,[fv]) — loan instalment
Monthly rate = annual% ÷ 12 | nper = years × 12
PPMT / IPMT
PPMT(rate,per,nper,pv) — principal portion of payment
IPMT(rate,per,nper,pv) — interest portion of payment
per = which period (e.g. 1 = first month)
For one-time deposit use pmt=0; for periodic payments use pv=0

Date Functions

TODAY / NOW / DAYS360
TODAY() — returns current date
NOW() — returns current date and time
DAYS360("8/1/2013","8/23/2013") → 22 (based on 360-day year)

Filtering

Auto Filter
Click anywhere in data → Data → Filter (arrows appear on headers)
Text Filter: Equals / Begins With / Contains / Does Not Contain
Number Filter: Greater Than, Between, Top 10 (use Bottom for least)
OR condition: select two values in same column's filter dialog
Remove filter: click Filter again OR Undo
Advanced Filter
AND — conditions on the same row of criteria range
OR — conditions on different rows of criteria range
Data → Advanced Filter → set List Range + Criteria Range
Can copy filtered results to another location
Unlike AutoFilter, Advanced Filter can impose conditions on multiple columns simultaneously

What-If Analysis

Goal Seek
Finds ONE input value for a target output
Data → Data Tools → What-If Analysis → Goal Seek
Set Cell = formula cell (must contain a formula)
To Value = target result
By Changing Cell = the variable to adjust
Click OK to apply; Cancel to revert
Limitation: only one changing variable at a time
Scenario Manager
Compare effect of multiple changing value sets
Data → What-If Analysis → Scenario Manager → Add
Give scenario a name → set Changing Cells → enter values
Add multiple scenarios (e.g. Original, Pay Hike 1, Pay Hike 2)
Summary → Scenario Summary → set Result Cells → OK
Widely used in budgeting to compare different expense/income scenarios

Workbook Linking

External References
Source workbook = where the data originates
Dependent workbook = contains link to source
Syntax: [SALARY.xlsx]Sheet1!B2
Any change in source auto-updates dependent workbook
Create by pointing: activate dependent cell → start formula → switch to source → click cell → Enter
Paste Link: Copy source cells → Paste Special → Paste Link in dependent workbook
Can use named ranges across workbooks: =BASIC!Salary*20%
